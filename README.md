# Gráfico com D3.js – Deixos

Projeto desenvolvido durante a pós-graduação em Desenvolvimento Full Stack.

## 📊 Descrição
Este projeto demonstra a utilização da biblioteca **D3.js** para criação de um gráfico com eixos personalizados, utilizando HTML, CSS e JavaScript puro.

## 🗂️ Estrutura do projeto
- `index.html` → estrutura da página
- `grafico.js` → lógica do gráfico com D3
- `grafico.css` → estilização do gráfico

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript
- D3.js

## ▶️ Como executar
Basta abrir o arquivo `index.html` no navegador  
ou utilizar a extensão **Live Server** no VS Code.

## 📚 Contexto
Atividade acadêmica focada em visualização de dados com D3.js.
